# VOLTLINK
VOLTLINK - Plataforma em Python para monitoramento de recarga e checkout automatizado em eletropostos. Projeto acadêmico para a disciplina de PISI 1 - UFRPE.

# VoltLink - Gestão de Eletropostos em Recife Sistema em Python para localização e
check-in em estações de recarga de veículos elétricos. 
###  🛠 Tecnologias e Bibliotecas - Python 3.x - SQLite3 (Banco de Dados) - OS e Time (Utilitários de sistema) 
### Como rodar o projeto 
1. Clone o repositório. 2. Certifique-se de
ter o Python instalado. 3. Execute o comando: `python main.py` 4. O banco de dados
será criado automaticamente no primeiro acesso.
### Funcionalidades - Cadastro de Usuário e Veículos. - Listagem Real de Postos em Recife-PE. - Sistema de Check-in e monitoramento de carga. - Área "Minha Conta" para gestão de perfil.
